MOBILE UX
-Design web apps for the smallest device first
-1. Be There
-2. Be Useful (Relevant)
-3. Be Quick (Simplicity)
-Reachability- how our users use their device (physically)- design accordingly
-Lots of people don’t have the top speeds, develop to ensure speed by:
1.Reduce image usage
2.SVG/Webfonts are lower bandwidth
3.Optimize and minify css and js files
4.GZIP files to eliminate unnecessary data
-Make the selections larger in forms for ease of selection. Reduce information to the bare necessity. Reduce cognitive load.
-If you require a lot of information, paginate the form (break it into sections). IF the form is large enough, consider adding a progress bar
-For passwords, default the passwords to not be hidden, but allow the user to hide it if they desire
-Use the number pad when the user needs to enter in numbers
-Link phone numbers so that they don’t have to copy and paste the number to use it
-Make mobile ads small and simple. Don’t use pop ups
-Don’t use pop-ups/pop-overs
-Make the ads unobtrusive, keep your content first
