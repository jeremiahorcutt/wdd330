CSS3 Transforms and Transitions-----------------
Transforms-----------------------------------------
The CSS3 transform property lets you lets you translate, rotate, scale, and/or skew any element on the page
Translation
Translation functions allow you to move elements left, right, up, or down. These functions are similar to
the behavior of position: relative; when declaring top and left , moving elements up and down or left
and right along the x and y axes.
transform: translate(45px, -45px);
Scaling
The scale(x,y) function scales an element by the defined factors horizontally then vertically. If only one
value is provided, it will be used for both the x and y values, growing or shrinking your element or
pseudo-element while maintaining the original aspect ratio
transform: translateX(40px) scale(1.5);
*Note that you shouldn’t declare a new transform : because of the cascade, a second transform would override the
first. To declare multiple transformations, provide a space-separated list of transform functions
Rotation
The rotate() function rotates an element around the point of origin by a specified angle value
transform: rotate(10deg) translateX(40px) scale(1.5);

Skew
The skew(x,y) function specifies a skew along the x and y axes
transform: skew(15deg, 4deg);
*The order of transform functions does matter: if you rotate before translating, your translate direction will be on the rotated axis
Transitions--------------------------------------
Declare the original state of the element in the default style declaration.
Declare the final state of your transitioned element; for example, a :hover state.
Include the transition functions in your default style declaration using the transition properties, including:
transition-property , transition-duration , transition-timing-function , and transition-delay
The transition-duration property sets how long the transition will take
The transition-timing-function lets you control the pace of the transition in even more granular detail
The most common timing functions include the key terms ease, linear, ease-in, ease-out, or ease-in-
Out
The transition-delay property makes it possible to introduce a delay before the transition begins
Longhand:
.ad-ad2 h1 span {
transition-property: transform;
transition-duration: 0.2s;
transition-timing-function: ease-out;
transition-delay: 50ms;
}
Shorthand:
.ad-ad2 h1 span {
transition: transform 0.2s ease-out 50ms;
}
Animations---------------------
A keyframe is a snapshot that defines a starting or end point of any smooth transition
@keyframes myAnimation {
/* put animation keyframes here */
}
the animation-timing-function determines how the animation will progress over its duration
The animation-iteration-count property lets you define how many times the animation will play through
The animation-delay property is used to define how many milliseconds or seconds to wait before the
browser begins the animation
.verbose {
animation-name: appearDisappear;
animation-duration: 300ms;
animation-timing-function: ease-in;
animation-iteration-count: 1;
animation-direction: alternate;
animation-delay: 5s;
animation-fill-mode: backwards;
animation-play-state: running;
}
/* shorthand */
.concise {
animation: 300ms ease-in alternate 5s backwards appearDisappear;
}

Canvas, SVG, and Drag and Drop------------------------------------------------------------------
Canvas---------
With canvas, we can draw shapes and lines, arcs and text, gradients and patterns. In addition, canvas
gives us the power to manipulate pixels in images and even video
‹canvas id="myCanvas" class="myCanvas" width="200" height="200"›
Sorry! Your browser doesn’t support Canvas.
‹/canvas›
the width and height attributes determine how large the canvas’s coordinate system is
var canvas = document.getElementById("myCanvas");
var context = canvas.getContext("2d");
context.strokeStyle = "red";
context.fillStyle = "blue";

SVG---------------
SVG stands for Scalable Vector Graphics, a specific file format that allows you to describe vector graphics
using XML

Both canvas and SVG allow you to draw custom shapes, paths, and fonts
Drag and Drop--------------------
This API allows you to specify that certain elements are draggable, and then specify what should happen
when these draggable elements are dragged over or dropped onto other elements on the page
